Ключевые функции
🗓️
Переход между годами
Приложение автоматически определяет текущий год. При переходе на новый год создаётся новый лист (напр. "1. Общая 2026"). Переключатель года виден в шапке.
✨
Умное автодополнение
Текстовые поля (ЧТО, ГДЕ, КОМУ) предлагают значения из предыдущих записей. Фиксированные значения (КАК, Оплата) — чипы для быстрого выбора.
🔄
Синхронизация с Google Sheets
Каждая новая запись добавляется в нужный месяц-блок таблицы. Чтение данных при загрузке страницы. Кэш в localStorage.
📊
Статистика
Графики доходов/расходов по месяцам (Chart.js). Сводка по каждому разделу. Фильтр по месяцу или весь год.
